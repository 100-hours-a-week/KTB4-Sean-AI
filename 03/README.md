# 3주차 회고
이번 주 배운 데이터 분석, 시각화 라이브러리 중 몇가지는 대학교에서 몇번 사용했던 기억이 있습니다. 그 때 당시에는 간단한 라이브러리 사용법만 익힌 후 원하는 분석값만 나오면 된다는 마인드로 잠깐 공부했는데, 이번주 강의를 들으면서 각 라이브러리들의 특징과 만들어진 이유를 알게되면서 조금 더 폭넓게 데이터 분석이라는 분야에 대해 이해하게 되었습니다. 아직 손에 완전히 익지는 않았지만 위클리 챌린지를 진행하면서 조금은 나아진 것 같습니다. 과제 이외의 시간으로는 공부 정리한 메모를 더 효율적으로 바꾸는데 시간을 많이 쓴 것 같습니다. 지난 2주간 메모정리는 폴더구조를 너무 복잡하게 해서 메모로 공부하는 시간보다 메모관리하는데 잡아먹는 시간이 너무 많았던 것 같습니다. 그래서 구조를 간단하게 바꾼 후 앞으로 정리한 메모는 깃허브에 매일 올리는 걸 목표함으로써 메모 정리하는 습관이 밀리지 않게 할 것 같습니다.
