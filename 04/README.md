# 4주차 회고
이번 위클리 챌린지는 이전보다 더욱 능동적으로 과제를 풀어보려고 노력했습니다. 본격적인 코드 작성에 앞서 문제별 접근 방식을 구체적으로 설계하고, 구현 후에는 개선점과 느낀 점을 기록했습니다. 이 과정을 통해 코드 한 줄 한 줄의 의미를 명확히 파악할 수 있었고, 각 모델의 장단점과 데이터셋에 대한 이해도를 한층 높일 수 있었습니다.

다만, 과제 후반부에는 시간 부족으로 7번 문제 코드 작성 시 AI의 도움을 받았으며, 이는 추후 코드를 면밀히 분석하며 보완할 계획입니다. 또한, 초기 접근을 Bottom-up 방식으로 진행하다 보니 구현 과정에서 시간이 지체되는 경향이 있었습니다. 다음 과제에서는 Alex가 추천해 준 Top-down 방식의 공부법을 적용해 효율성을 높여보려 합니다.

아직 AI 튜터를 100% 활용하지 못했다는 아쉬움이 남아, 다음 주에는 AI 튜터를 적극적으로 사용해 보고 나에게 가장 효과적인 활용 방안을 찾아 기록해 둘 예정입니다.
